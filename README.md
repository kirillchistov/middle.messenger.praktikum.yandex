# Chat App - Yandex Practicum Middle Front-end project

Современная посадочная страница для сервиса аналитики рекламных кампаний Points Track, построенная на React 19 с TypeScript.

## 🎯 Технологии, использованные в проекте

- ⚛️ Языки **JavaScript** и [**TypeScript**]() - базовые инструменты для реализации приложения. Версия TypeScript ≥ 3.8.
- 🎨 [**PostCSS**]() — для описания внешнего вида.
- ⚡ [**Netlify**]() — сервис для разворачивания приложения в сети интернет.
- 🚀 [**Vite**]() — инструмент для автоматизации ручной сборки приложения.
- ⚡ **ESlint**, **Stylelint**, **.editorconfig** — инструменты для регулирования определённого и единственного стиля кода JS, *CSS-файлов.
- 📱 **Шаблонизаторы** — инструменты для переиспользования уже готовых компонентов.
- ⚡ **Chai**, **Mocha**, **Jext** — инструменты для написания тестов.

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/kirillchistov/middle.messenger.praktikum.yandex.git
cd middle.messenger.praktikum.yandex
```

### 2. Установка зависимостей

```bash
pnpm install
```

### 3. Локальная разработка

```bash
pnpm dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.
